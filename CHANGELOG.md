# Changelog - Museo Bucolicò

## [1.0.0] - 2026-05-08

### Aggiunto
- Inizializzazione progetto su Next.js 15 (App Router) e React 19.
- Styling system tramite Tailwind CSS v4 con design token personalizzati.
- Supporto a TypeScript per type-safety.
- Componente `SeasonalHero` con background immersivo e tipografia elegante.
- Componente `TimelineNarrative` con layout che sfuma simulando le 4 stagioni.
- Componente `ArtifactGrid` per la visualizzazione suggestiva ed evocativa della collezione.
- Componente `CommunitySection` con collage asimmetrico per lo storytelling.
- Call To Action in `VisitInfo` e `Footer`.
- Supporto accessibilità: font leggibili e dark mode contrastata.
- Ottimizzazione SEO tramite metadata integrati in layout.tsx.
- Dockerizzazione (Dockerfile standalone multi-stage + docker-compose.yml).
- Documentazione completa (README.md, DEPLOYMENT.md, CONTENT_GUIDE.md).

### Migliorato
- Migrazione totale dell'identità visiva e dei testi dal vecchio sito test (HTML base) alla nuova struttura a singola pagina ad altissime prestazioni.
